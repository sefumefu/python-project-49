# Brain Games

Набор игр для тренировки мозга: арифметика, чётные числа, НОД, простые числа и прогрессии.

## Минимальные требования

- Python >= 3.13
- uv

## Установка

```bash
uv tool install git+https://github.com/sefumefu/python-project-49
```

## Запуск

```bash
brain-games
brain-calc
brain-even
brain-gcd
brain-prime
brain-progression
```
